# combine-spliter

Combines and splits text files
```
usage: combine-spliter [options] path [filename]
Flags:
  -combine
    	Combine into single file
  -lines int
    	Lines per file
  -pattern string
    	Glob pattern to use (default "*")
  -skipLines int
    	Lines to skip per file
  -split
    	Split single file
  -subDir
    	Inlcude sub directories in combine (default true)
```
